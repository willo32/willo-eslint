# eslint-willo-config-react

ESLint plugin to enforce Willo coding rules for React

## Installation

```sh
$ yarn add --dev eslint-config-willo-react
```

## Usage

### Via `.eslintrc.js` (Recommended)

```js
module.exports = {
    extends: ['eslint-config-willo-react'],
    rules: {
        // My Rules
    },
};
```
