# eslint-config-willo-base

ESLint plugin to enforce Willo coding rules for ES6

## Installation

```sh
$ yarn add --dev eslint-config-willo-base
```

## Usage

### Via `.eslintrc.js` (Recommended)

```js
module.exports = {
    extends: ['eslint-config-willo-base'],
    rules: {
        // My Rules
    },
};
```
